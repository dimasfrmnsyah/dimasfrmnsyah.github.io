# Premium Portfolio

A modern, premium personal portfolio built with Next.js (App Router), TypeScript, Tailwind CSS, and Framer Motion.

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` – start development server
- `npm run build` – build for production
- `npm run start` – start production server
- `npm run lint` – run ESLint

## Customize

Update your content in:

- `data/hero.ts`
- `data/projects.ts`
- `data/skills.ts`
- `data/experience.ts`
- `data/education.ts`
- `data/socials.ts`
- `data/stats.ts`

Replace assets in `public/images` and update `public/cv.pdf` with your real CV.
